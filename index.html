The fast path — do this tonight

Chat AI → OverleafBrowser only · ~15 min first time
You’ll need: any chat-based AI — Ask Sage (the DoD-authorized, CUI-approved one), or ChatGPT / Claude / Gemini for non-sensitive memos — and a free Overleaf account. Plus the starter kit above (just unzip it). For sensitive (CUI) memos, use asksage.ai (Army CAC users sign in at chat.genai.army.mil). For routine memos, any chatbot you already have works.
Part A · Write it (in the chat)
  1. Open your AI (Ask Sage, ChatGPT, or Claude) and start a new chat. If it asks you to pick a model, choose the newest one.
  2. Copy the prompt below and paste it into the chat. (It already has the formatting built in, so you don’t upload anything here.)
  3. Replace the words inside the brackets with your details, but keep the \backslash words exactly as they are. Easiest: leave the <…> bits alone and just fill the [ brain-dump ] block at the bottom — the AI fills in the rest. Then send.
  4. The AI replies with a block of formatting text. Copy only that text — it should start with \documentclass. If it added ``` marks or sentences around it, leave those out.
Part B · Get the PDF (Overleaf)
  1. Unzip the starter kit you downloaded above — you’ll upload two files from it.
  2. Go to overleaf.com, sign up free, then click the green New Project → Blank Project, give it any name, and click Create. (picture 1 below)
  3. In the file panel on the left, click the upload icon (an up-arrow) and add armymemo.cls and DODb1.pdf from the kit folder. (picture 2)
  4. The middle area shows a file called main.tex with sample text. Select all of it, delete it, and paste in what you copied from the AI.
  5. Open Settings (the gear icon) → Compiler → choose LuaLaTeX — the default won’t work here. Then click the green Recompile. (picture 3)
  6. Click Download PDF. That’s your memo.
  7. Blank lines? An underline shows wherever the AI didn’t have a detail. In the text, find the matching \rule{…}, type your information over it, and Recompile.
Part A — paste this into the chat
You are a military correspondence specialist. Write an Army Memorandum for Record that strictly complies with AR 25-50.

Output it as LaTeX source using the `armymemo` document class, matching this exact structure (keep these commands, fill in the values):

    \documentclass{armymemo}
    \address{<organization name>}
    \address{<street address>}
    \address{<city, state  ZIP+4>}
    \officesymbol{<office symbol>}
    \signaturedate{<day Month year>}
    \memoline{MEMORANDUM FOR RECORD}
    \subject{<subject line, 10 words or fewer>}
    \author{<FULL NAME IN CAPS>}\rank{<rank spelled out>}\branch{<branch abbrev>}
    \title{<duty title>}
    \begin{document}
    \begin{enumerate}
    \item References.  <authorities, or omit if none>
    \item Purpose.  <bottom line up front, 1-2 sentences>
    \item <body paragraph(s); use nested \begin{enumerate} for subparagraphs>
    \item The point of contact is the undersigned at <phone> or <email>.
    \end{enumerate}
    \end{document}

Writing rules:
- Active voice throughout. Bottom line up front in paragraph 2 (Purpose).
- References in paragraph 1, point of contact in the last paragraph.
- Subject line: 10 words or fewer. Two spaces after each period.

CRITICAL — do not invent any fact. If you do not have a name, date, unit, phone number, or statistic, leave a blank exactly like this: \rule{1.5in}{0.4pt} — never make one up.

OUTPUT FORMAT — return ONLY the raw LaTeX, starting with \documentclass{armymemo}. Do NOT wrap it in markdown code fences (no triple backticks) and do NOT add any explanation before or after it.

Here is what the memo is about:
[ Brain-dump everything you know here — topic, your name/rank/unit if you want it filled in, the policy or event, key dates, who it's for. Messy is fine. ]

Two non-negotiables

1. Mind where the words go. Ask Sage is a government, CUI-approved system — fine for sensitive memo content. Overleaf and commercial chatbots are not. Never paste CUI, PHI, or real PII into them. If your memo contains CUI, skip Overleaf — make the PDF on your own computer instead (below), so nothing leaves your machine.

2. You sign it, you own it. AI invents names, dates, and regulation numbers with total confidence — a wrong reg number looks exactly as authoritative as a right one. The sample prompt tells it to leave blanks instead of guessing. Read every line before it goes out; “the AI wrote it” is not a defense.

Optional — when you’re ready to go faster

OptionalThe fast path above is a complete, permanent solution — many people never need more. If you make memos often, here are two setups that remove the copy-paste.

First, the one bit of jargon. That block of formatting text has a name — LaTeX (say “LAY-tek”). Overleaf is just a free website that turns it into the finished PDF. And the reason every memo comes out in correct AR 25-50 format is one file in the kit, armymemo.cls — think of it as the formatting rulebook. You and the AI supply the words; it handles the layout. The two options below do the same thing, just without copying text between two websites.

Claude Cowork

An app on your computer · writes the file for you
Need: the Claude desktop app (Cowork, on any paid plan — Pro or higher).
  1. Put the starter-kit folder in your Documents folder.
  2. Open the app, click Cowork, point it at that folder, grant access.
  3. Tell it: “Using the armymemo template here, write an MFR about [your facts]. Save it as a new .tex file. Don’t make anything up — leave blanks.”
  4. It writes the file for you. For the PDF, hand the file to Overleaf, or make it on your computer if you’ve installed the software (below).

Claude Code in VS Code

The way I do it in class · most setup, least effort after
Need: VS Code, Claude Code, and LaTeX installed (below).
  1. Open a folder containing armymemo.cls + DODb1.pdf in VS Code; start Claude Code.
  2. Ask in plain English: “Using armymemo.cls here, write an MFR about [facts], leave blanks for anything missing, then compile twice with lualatex and open the PDF.”
  3. It writes the memo, turns it into a PDF, and opens it — no copy-paste, and nothing leaves your machine.
  4. Iterate by talking. (In class I use a custom /memo skill that also auto-loads the regs; you can build one once you’re comfortable.)
Making the PDF on your own computer (needed for both options, and for any CUI memo): this replaces Overleaf with free software on your machine. Install MacTeX (Mac) or TeX Live (Windows/Linux) — a one-time download, a few GB. Keep armymemo.cls and DODb1.pdf next to your memo file, then run this command twice: lualatex memo.tex && lualatex memo.tex. Because nothing leaves your computer, this path is safe for CUI. (The memo uses Times New Roman + Arial; Mac and Overleaf already have them, a bare Linux machine may need the msttcorefonts package first.)
